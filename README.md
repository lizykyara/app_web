# Desenvolvimento Web - Cadastro de Currículos :memo:

Projeto simples de desenvolvimento web para cadastramento de currículos.
O usuário pode informar seu nome completo, e-mail, telefone e anexar seu currículo.

# Ferramentas usadas

Python
Flask
